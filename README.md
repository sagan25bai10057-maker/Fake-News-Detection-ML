# Fake-News-Detection-ML
Machine Learning project to detect fake news.

# 📰 Fake News Detection using Machine Learning

## 📌 Project Overview
This project is a Machine Learning-based system that classifies news as **REAL** or **FAKE** based on textual content.

The model is trained using a dataset of real and fake news articles and uses Natural Language Processing (NLP) techniques for classification.

---

## 🎯 Problem Statement
With the rapid spread of misinformation online, it has become difficult to identify whether a news article is genuine or fake. This project aims to solve this problem using machine learning.

---

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
- NLP (TF-IDF Vectorization)

---

## 🧠 How It Works
1. Load dataset (Fake & Real news)
2. Clean text (remove punctuation, links, etc.)
3. Convert text into numerical form using TF-IDF
4. Train model using SGD Classifier
5. Predict whether input news is REAL or FAKE

---

## 📊 Model Performance
- Accuracy: ~99%
- Includes Confusion Matrix for evaluation
- Confidence score added for prediction reliability
